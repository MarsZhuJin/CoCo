# CoCo

![License](https://img.shields.io/dub/l/vibe-d.svg)

CoCo is a Code Convert tool which can transform file's encoding format.


## Install

#### install by pip
```
pip install cocov
```



#### install by source
```
git clone git@github.com:buptmiao/CoCo.git  
cd CoCo
python setup.py install


```
